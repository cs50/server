# Usage

|File|Inferred application type
|----|-------------------------
|`app.js`|Node.js or Meteor JS in bundled/packaged mode
|`config.ru`|Ruby, Ruby on Rails
|`.meteor`|Meteor JS in non-bundled/packaged mode
|`passenger_wsgi.py`|Python
|`*`|HTML, PHP
